    Argus

    Argus is a tool that fetches information about people from Wikipedia by parsing infoboxes and category information stored in wikipedia dumps. The collected data is then cleaned using various methods (see paper for further description). A purpose-built web application was also made which enable users to post queries to the collected database.
    Paper (pdf) · Poster describing the process (pdf)
    Keywords: Information retrieval, Python, Java, CSS, XML

    Blopper

    Blopper is a Virtual Reality balloon popping simulator, with kinect body tracking.
    The user wears a Google Cardboard Virtual Reality headset, showing them a 3d world that can be navigated.
    The player is being tracked in real time by a Kinect motion sensor,
    enabling quick and wire-free interaction with the environment.

    Blopper was presented with great success at the 2015 Comic con Gamex gaming convention.
    Short video preview
    Keywords: C#, Kinect, Google Cardboard, Unity

    Oculus Rift Molecule Visualizer

    A visualization of molecular data, using an Oculus Rift and a gamepad controller.
    Molecules stored in the Protein Data Bank (.pdb) file format are loaded into a 3d world (using a custom made parser written in Javascript), which the user can explore in full 3d in a way that would be hard to do on a 2d screen.
    Keywords: C#, Javascript, Unity, Oculus Rift

    O-chat

    O-chat is a multi-threaded Java chat program. It supports chatrooms,
    file transfers, UTF-8 compatibility and AES/Blowfish/RSA encryption (messages and files).
    It eatures a command line interface as well as a standard graphical user interface.
    Code on github: O-chat
    Keywords: Java, encryption

    Tamapro

    Tamapro is a multiuser application inspired by the Tamagotchi brand of toys.
    A centralized server keeps track of multiple monsters (called Tamas, owned by different users), which interact with each other.
    Users connects to the server (written in Python with the Bottle library) using either an Android application or a web interface.
    All information about monsters are stored in a SQLite3 database.
    Github repository for the server
    Github repository for the Android client Keywords: Python, Java, Android, SQL
